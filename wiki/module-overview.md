# Module Overview

The Module is designed to deploy External DNS on EKS clusters using Terraform. It includes resources for setting up the necessary Pod Identity and Helm release for External DNS. The module is structured to allow for easy integration into existing Terraform configurations, with clear input variables and output values. It also allows for account specific configuration through the use of Terraform workspaces, enabling different settings for different environments (e.g. development, staging, production). The module follows the Ministry of Justice GitHub Repository Standards and is licensed under the MIT License.
